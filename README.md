# DayBook

DayBook is a small SwiftUI diary app for macOS with a calm, reflective feel.

It lets you:

- write a diary entry for a selected day
- seal entries so they become read-only
- browse past entries in a calendar view
- export archived entries when needed

Notes:

- diary entries are stored locally on your machine
- this repository contains the app source code, not your personal diary data

The app is designed to feel quiet and personal: soft retro-inspired visuals, a simple writing flow, and a focus on keeping each day preserved once it is sealed.

Built with Swift, SwiftUI, and Swift Package Manager.
